# 使用方法
#### node接入微信公众号开发
#### npm install 
#### npm start
#### 启动一个ip外网映射工具（例 natapp）[natapp](https://natapp.cn/) 
#### 在微信里配置natapp获得的备案过的域名
#### 然后关注微信号进行测试
