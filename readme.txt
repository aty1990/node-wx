# 使用方法
##### 1 npm install 
##### 2 npm start
##### 3 启动一个ip外网映射工具（例natapp）![natapp](https://natapp.cn/) 
##### 4 在微信里配置natapp注册的得到的`备案`过多的域名
##### 5 最后关注微信号进行测试
